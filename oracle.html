terminology

ITL

ITL (Interested Transaction List) is an array of 23-byte entries in the variable portion of an Oracle data block. Any process that changes data, must store it's transaction id and undo segment address in an ITL.

Undo block

For read consistency we make a copy of the data block in memory and apply the undo records to that block, and it's a copy of the block that we can discard very rapidly once we've finished with it;

THere is an important principle of optimization that is often overlooked. Sometimes it is better for everyone to do a little more work if that means they are operatiing in separate locations rather than constantly colliding on the same contention point-competition wastes resources.
